LaTeX 入门

LaTeX 是一个强大的排版系统，尤其适合撰写包含大量数学公式、图表和结构化内容的文档。本文将简要介绍如何使用 LaTeX 编写数学公式以及插入浮动体（图表等）。

1. 基本数学公式

数学公式分为行内公式与块级公式，分别使用 $...$ 与 \[...\] 来编写：

行内公式：

例如：$a^2 + b^2 = c^2$ 表示勾股定理。

6. 小贴士

  • 使用 \text{} 在公式中插入文本，例如：$x > 0 \text{ 时有解}$。
  • 使用 \left 和 \right 自动调整括号大小，例如：$\left( \frac{a}{b} \right)$。
  • 控制空格：\quad, \,, \! 等。

7. 编译推荐工具

  • 本地：TeX Live + VSCode（推荐插件 LaTeX Workshop）
  • 在线：Overleaf（适合协作）
